# Diabetes-Prediction-ML
Machine Learning model to predict diabetes using Logistic Regression 
#  Diabetes Prediction using Machine Learning

Project Overview

This project aims to predict whether a patient has diabetes based on medical attributes using Machine Learning.
The model is trained on the Pima Indians Diabetes dataset and uses Logistic Regression for classification.



Technologies Used

* Python 
* NumPy
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn



Dataset

* Dataset: Pima Indians Diabetes Dataset
* Features include:

  * Glucose level
  * Blood pressure
  * BMI
  * Insulin
  * Age

 Model Used

* Logistic Regression (Classification)

Results

* Accuracy: 77.79%
* Evaluation Metrics:

  * Confusion Matrix:
    [[89 10]
     [24 31]]

How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/dipjyotiroy158-coder/Diabetes-Prediction-ML
   ```
2. Navigate to the folder:

   ```
   cd Diabetes-Prediction-ML
   ```
3. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
4. Run the Jupyter Notebook:

   ```
   jupyter notebook
   ```
